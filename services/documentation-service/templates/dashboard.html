System Architecture Map

Online Stack

Clients communicate exclusively with the API Gateway via GraphQL. The Gateway handles validation and delegates requests using gRPC to independent backend microservices, which run on dedicated databases. High-throughput notifications and search index rebuilds run asynchronously via NATS JetStream.
